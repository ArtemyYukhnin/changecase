# Конвертер регистров
Учебный проект.

Функции, позволяющие менять регистр символов(английского алфавита) в входящем файле в зависимости от входящего параметра.
- ВЕРХНИЙ РЕГИСТР на нижний регистр;
- нижний регистр на ВЕРХНИЙ РЕГИСТР;
- регистры в случайном порядке;
- или переключать регистры.

## Командная строка
```
  changecase -key source_filename destination_filename
```

Поддреживаемые ключи:
- `-U` - преобразование текста в ВЕРХНИЙ РЕГИСТР;
- `-L` - преобразование текста в нижний регистр;
- `-G` - переключение регистров (из ВЕРХНЕГО в нижний и наоборот);
- `-T` - каждое слово начинается с заглавной буквы, остальные же буквы в нижнем регистрe;
- `-R` - преобразование каждого символа в случайном регистре.
